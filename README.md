# 📊 Data Manager and ADTs in Python

This repository contains the structured implementation of Abstract Data Types (ADTs) developed as an academic project for the University of Murcia (Bachelor’s Degree in Mathematics and Physics).

## 🎯 Project Objective
The main objective was to build complex data structures at a low level, avoiding the use of Python’s native collections (such as dictionaries or sets).

## 🛠️ Main Features
* **Dataframe ADT:** Creation of a custom tabular structure for data manipulation and analysis.
* **Mail Manager ADT:** Implementation of management and filtering logic.
* **Memory Management:** Efficient and structured design using Object-Oriented Programming.
